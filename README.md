# 🩺 Sistema de Gestão de Pacientes (API REST)

Uma aplicação Full Stack desenvolvida do zero para resolver uma demanda real de gestão de contatos clínicos, permitindo o cadastro, leitura, atualização, exclusão (CRUD) e exportação de dados para Excel.

## 🚀 Tecnologias Utilizadas
- **Back-end:** Node.js, Express
- **Banco de Dados:** SQLite (Relacional)
- **Front-end:** HTML5, CSS3, Vanilla JavaScript
- **Testes Automatizados:** Test runner nativo do Node.js (`node:test`)
- **Utilitários:** SheetJS (Exportação para `.xlsx`)

## ⚙️ Funcionalidades
- Cadastro de novos pacientes (Nome e WhatsApp).
- Listagem em tempo real na interface gráfica.
- Atualização e Exclusão de registros.
- Exportação completa do banco de dados para planilha Excel.
- Cobertura de testes automatizados garantindo a integridade da API.
